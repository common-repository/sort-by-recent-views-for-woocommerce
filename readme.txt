== Description ==

If you have a WooCommerce store, Sort By Recent Views For WooCommerce is a FREE plugin that allows you to add the "Sort by recent views" sorting option to your shop.

== Features ==

- Add the "Sort by recent views" sorting option to your shop.
- Choose the sorting option title.
- Choose a specific tracking timeframe window (7, 14, 30, or 90 days).
- Automatically daily maintenance of the views database table.

== Installation ==

1. Unzip the download package
2. Upload `sort-by-recent-views-for-woocommerce` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Please note that the plugin will only start tracking product views from the moment it is activated. In addition, the view counts are recalculated once every hour, so the "Sort by recent views" sorting option will only start showing real results 1 hour after activating the plugin.
